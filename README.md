# Michelle Johnson Productions

Official static website for Michelle Johnson Productions.

## Pages
- `index.html` — portfolio, services, featured experiences, testimonials, and contact
- `insights.html` — thought-leadership hub for Cue to Client™ and related articles
- `assets/` — production photography and testimonial graphics

## Deployment
This site is designed for direct deployment to Vercel as a static website. No build command is required.
